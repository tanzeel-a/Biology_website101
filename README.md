# Git_Push_Biology

A visually striking, fully animated biology-themed website exploring the essence of life.

## Features

*   **Immersive Hero Section**: Dark, neon-themed biology background with particle effects and typewriter animations.
*   **Interactive Timeline**: A scroll-triggered timeline showcasing major biological discoveries from the Cell to CRISPR.
*   **Knowledge Hub**: Curated links to top biological resources like Nature, Science, and NCBI.
*   **Responsive Design**: Fully optimized for desktop and mobile viewing.

## Technologies Used

*   **HTML5**: Semantic structure.
*   **CSS3**: Custom properties, flexbox/grid, animations, and responsive design.
*   **JavaScript**: `IntersectionObserver` for scroll animations, dynamic text typing, and Particles.js integration.

## Setup

1.  Clone the repository:
    ```bash
    git clone https://github.com/tanzeel-a/Biology_website101.git
    ```
2.  Open `index.html` in your web browser.
